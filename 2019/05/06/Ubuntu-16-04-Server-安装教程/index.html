Ubuntu 16.04 Server 安装前准备

1.下载镜像文件

Ubuntu官网 -> Download -> Alternative downloads -> Past release -> 16.04 -> 64-bit PC (AMD64) server install image

一步到位：
官网 Ubuntu-16.04-server-amd64.iso
清华镜像 Ubuntu-16.04-server-amd64.iso
华为镜像 Ubuntu-16.04-server-amd64.iso

  3. 选择语言，English

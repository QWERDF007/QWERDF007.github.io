Mat Public Attributes

  • cols：列数，图像宽度
  • rows：行数，图像高度
  • dims：维度

Mat Public Member Functions

  • at：模板函数，返回对指定元素的引用
  • begin & end：配合迭代器使用
  • channels：返回矩阵的通道数
  • clone：返回矩阵的副本
  • copyTo：将矩阵拷贝至指定矩阵
  • create：创建矩阵
  • empty：判断矩阵是否为空
  • dot：矩阵乘法
  • ptr：模板，指向指定行的指针
  • diag：创建对角矩阵
  • eye：创建单位矩阵
  • ones：创建全1的矩阵
  • zeros：创建全0的矩阵
